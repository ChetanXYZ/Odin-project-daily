HTML Entities

There are three reserved characters in HTML: < > (greater than and lesser than symbol) and & (and symbol). You should always use HTML entities for these three characters

  1. “
  2. ”
  3. ‘
  4. ’

If you’re into “web typography,” you’ll also find yourself using curly quote quite a bit.
